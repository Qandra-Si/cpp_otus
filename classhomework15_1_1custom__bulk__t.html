Кастомный планировщик команд

Имеет реализацию интерфейса abstract_bulk_t. Считает сохранённые (отложенные) для выполнения команды и когда их накапливается заданное количество - обращается к методу run экземпляра класса commands_processor_t. Отслеживает групповые операции (транзакции) с помощью которых подготавлиется единая группа команд до тех пор, пока не будут закрыты все вложенные транзакции, - почле чего также вызывает метод run обработчика команд.
