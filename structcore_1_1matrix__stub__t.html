struct core::matrix_stub_t< T >

"Заглушка" шаблонизируемого параметра, реализующая copy/direct initialization

Предназначена для перегрузки оператора тип() по умолчанию для приведения типа по умолчанию при чтении значений из "заглушки". Имеет интерфейс изменения данных, ассоциированных с "заглушкой", вызывает callback при обращении к assignment оператору.
