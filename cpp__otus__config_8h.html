Конфигурация платформы

Файл <cpp_otus_config.h>

Определяются следующие платформы:

  • Linux - макрос CPP_OTUS_LINUX
  • BSD - макрос CPP_OTUS_BSD
  • Solaris - макрос CPP_OTUS_SOLARIS
  • QNX - макрос CPP_OTUS_QNX
  • MacOS - макрос CPP_OTUS_APPLE
  • Unix (неизветная версия Unix системы) - макрос CPP_OTUS_UNINX
  • Windows - макрос CPP_OTUS_WINODWS
  • Cygwin - макрос CPP_OTUS_CYGWIN

Правила конфигурации платоформы:

  • макрос CPP_OTUS_PLATFORM содержит название (строковое представление) платформы
  • макрос CPP_OTUS_POSIX определяем является ли данная ОС истиной(правильной) и подерживает POSIX (все платформы кроме Windows)
Заметки
Если платформа определна как Cygwin - то макрос CPP_OTUS_WINDOWS НЕ доступен, однако определен макрос CPP_OTUS_POSIX

Конфигурация компилятора

Файл <cpp_otus_config.h>

Определяются следующие компиляторы:

  • GCC (включая MinGW) - макросы CPP_OTUS_CXX_GCC, CPP_OTUS_CXX_GCC_VER
  • LLVM clang - макросы CPP_OTUS_CXX_CLANG, CPP_OTUS_CXX_CLANG_VER
  • Bolrand - макросы CPP_OTUS_CXX_BORLAND, CPP_OTUS_CXX_BORLAND_VER
  • MS Visual C - макросы CPP_OTUS_CXX_MSCm CPP_OTUS_CXX_MSC_VER

Макрос CPP_OTUS_CXX содержит название (строковое представление) компилятора

Макросы экспорта/импорта функци

Файл <cpp_otus_config.h>

Определены следующие макросы облегчающие жизнь при переносе кода между платофрмами:

  • CPP_OTUS_SYMBOL_EXPORT - макрос определяет экспортируемые ф-ции для Windows библиотек
  • CPP_OTUS_SYMBOL_IMPORT - макрос определяет импортируемые ф-ции для Windows библиотек
  • CPP_OTUS_STDCALL - соглашение о вызове __stdcall
  • CPP_OTUS_CDECL - соглашение о вызове __cdecl
